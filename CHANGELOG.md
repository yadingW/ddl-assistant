# Changelog

## v1.1.0

- 优化首次打开体验：更新输入框 placeholder 示例，并区分两种无提醒空状态。
- 支持复制单条已完成事项。
- 支持按任务计划日期复制本周和上周已完成事项；复制周次不使用 `completedAt`。
- 改为固定的上下布局，记录区约 55%、提醒区约 45%，两区均支持内部滚动。
- 压缩记录区头部与工具区，保留主输入框最低可用高度；长线事项展开后改由记录区内部滚动承接。
- 优化主页面 DDL 底部历史存档区布局，使“查看历史已完成”和“复制本周已完成”同行显示，并与顶部复制入口列对齐。
- 整理窄窗口单列布局、任务行操作和空状态排版。
- 更新前端资源版本和 Service Worker 缓存版本，确保 GitHub Pages 与 PWA 获取新代码。

## v1.0.1

- 修复右侧任务反向定位原文时对长行自动换行、窄窗口和文本末尾位置不准确的问题。
- 点击任务正文后会先显示对应编辑区，再滚动文本框并定位到目标行。
- 已完成任务前台只显示“已完成”，不再显示“处理于 HH:mm”。
- 继续保留 `completedAt` 作为用户标记完成时间戳，用于历史排序、归档和旧备份兼容；它不是经过核验的实际完成时间。
- 更新前端资源版本和 Service Worker 缓存版本，确保 GitHub Pages 与 PWA 获取新代码。

## v1.0.0

- 发布连续工作随记、长线事项备忘、日期和时间解析、待跟进事项、完成状态、历史归档、PWA 离线缓存、本地提醒和 JSON 备份恢复。
