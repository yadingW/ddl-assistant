# 工作便签本 (Work Notepad)

一款面向办公室知识工作者的极简便签与 DDL 管理器。以“纯文本记录”为绝对核心，采用“输入即提取”的静默解析机制。没有任何复杂的任务表单、标签或优先级选项，完全契合日常办公中最本能的随记习惯。

本项目为纯前端架构（HTML/CSS/JS），无后端服务，数据完全保存在本地浏览器（LocalStorage），并支持作为 PWA 桌面应用独立安装。

## 核心设计理念

- **优先记录，其次整理，最后才是任务**：左侧 70% 的纯文本区域是唯一的事实来源（Single Source of Truth），右侧 30% 的 DDL 列表只是基于文本派生的辅助提醒视图。
- **状态与文本解耦**：右侧的任务勾选状态独立保存，绝不污染或修改左侧的原始工作笔记。

## 主要功能特性

### 1. 智能自然语言解析引擎
- **相对日期绝对化（输入即编译）**：输入“明天”、“后天”、“下周二”，系统会自动将其转化为绝对日期（如 `6.16`）并替换原文，防止日期跨日滚动失效。
- **上下文前缀继承**：以冒号（`:` 或 `：`）结尾的文本行（如 `A项目交付：`），会自动作为下方任务的标题前缀。
- **日期作用域继承**：单独书写日期的行（如 `6.15`），其下方的文本即使不带日期，也会自动继承该 DDL。
- **极简备注截断**：使用双斜杠 `//` 作为截断符。符号前的内容进入右侧 DDL 面板，符号后的内容仅作为笔记保留在左侧。
- **深度视觉降噪**：系统在生成右侧任务标题时，会自动剔除触发日期的原词（如“下周一”）、排版序号（如 `(1)`、`1.`）以及多余的破折号，保持列表极致清爽。

### 2. 双轨数据流转机制
- **按周分片（动态流）**：主输入区按自然周自动分片（如 `W25`）。每周一自动提供一张干净的新便签。支持随时呼出并编辑上一周的历史记录。
- **长线备忘（静态流）**：底部常驻“长线项目与商务节点备忘”抽屉，数据永久保留。系统会同时扫描这两个区域，将所有 DDL 统一汇聚到侧边栏。

### 3. 高效交互与灾备
- **反向定位机制**：在右侧面板点击任意任务，左侧便签会自动展开对应区域、滚动居中，并将光标精准定位到该行末尾，方便快速涂改延期。
- **本地数据灾备**：底部提供极简的 `[导出备份]` 和 `[恢复备份]` 功能。一键将所有随记、长线节点和任务状态打包为 JSON 文件，彻底解决清理浏览器缓存导致的数据丢失风险。
- **自由涂改顺延**：未完成的任务如果需要延期，直接在左侧文本中将日期改为明天，右侧系统会自动更新状态，符合物理便签的涂改直觉。

## 安装与使用指引

### 本地测试运行
如果你只需临时查看：
直接双击 `index.html` 在浏览器中打开即可使用基本功能。

### 作为 PWA 独立应用安装（推荐）
为了获得最佳体验并将其固定在任务栏，必须通过本地服务器运行：
1. **VS Code 用户**：安装 `Live Server` 插件，右键点击 `index.html` 选择 "Open with Live Server"。
2. **Python 环境用户**：在项目文件夹打开终端，运行 `python -m http.server 8000`（Mac 使用 `python3`），然后在浏览器中访问 `http://localhost:8000/`。
3. 页面打开后，点击浏览器地址栏右侧的“安装应用”图标，即可作为桌面端小程序独立运行。

## 目录结构
- `index.html`：极简主界面结构。
- `style.css`：UI 样式，包含 70/30 布局及视觉克制规范。
- `app.js`：核心业务逻辑（按周存储、正则解析、防抖提取、容灾导入导出）。
- `manifest.json` & `sw.js`：PWA 离线缓存与应用注册配置。
