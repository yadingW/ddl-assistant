# DDL 助手

一个纯前端的 DDL 任务整理网页应用，支持从 Excel 或普通文本中粘贴任务内容，自动识别日期并按到期时间分类。项目使用 HTML、CSS、JavaScript 实现，不依赖后端服务，数据保存在浏览器 LocalStorage 中。

## 项目简介

DDL 助手适合日常办公场景，用于快速整理带日期的待办事项。用户可以从 Excel 表格、邮件、聊天记录或普通文本中复制内容并粘贴到页面中，系统会自动提取可识别的日期作为 DDL，并生成任务列表。

## 功能说明

- 支持粘贴 Excel 表格内容，自动按行解析任务。
- 支持直接粘贴普通文字。
- 自动识别常见日期格式，例如：
  - `2026-06-08`
  - `2026/06/08`
  - `2026年6月8日`
  - `6/8/2026`
  - `6月8日`
- 自动分类任务：
  - 今日到期
  - 本周到期
  - 未来任务
- 支持任务完成 / 未完成状态切换。
- 支持删除单个任务。
- 支持清除已完成任务。
- 支持按全部、未完成、已完成筛选任务。
- 使用浏览器 LocalStorage 保存数据，刷新页面后任务仍会保留。
- 页面简洁，适合办公使用。

## 使用方法

1. 直接打开 `index.html` 文件。
2. 将 Excel 表格内容或普通文字粘贴到输入框中。
3. 点击“添加任务”。
4. 页面会自动识别每行中的日期，并按到期时间分类显示。
5. 勾选任务左侧复选框可切换完成状态。
6. 点击任务右侧删除按钮可删除单个任务。
7. 点击“清除已完成”可批量移除已完成任务。

示例输入：

```text
提交周报 2026-06-08
项目复盘	2026/06/12	张三
6月15日 完成合同审核
```
